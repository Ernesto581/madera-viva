# MaderaViva 🌲

Guía interactiva sobre tipos de madera, técnicas de carpintería y herramientas esenciales.

## Tecnologías

- **React 18** + **TypeScript**
- **Vite** como bundler
- **Tailwind CSS** para estilos
- **shadcn/ui** como librería de componentes
- **React Router** para navegación
- **Framer Motion** para animaciones
- **Recharts** para visualización de datos

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/Ernesto581/wood-wisdom-hub.git

# Entrar al directorio
cd wood-wisdom-hub

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

La app estará disponible en `http://localhost:8080`.

## Scripts disponibles

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Servidor de desarrollo |
| `npm run build` | Build de producción |
| `npm run preview` | Preview del build |
| `npm run lint` | Ejecutar ESLint |
| `npm run test` | Ejecutar tests |

## Estructura del proyecto

```
src/
├── components/     # Componentes de la UI
│   └── ui/         # Componentes base (shadcn/ui)
├── data/           # Datos de maderas
├── hooks/          # Custom hooks
├── lib/            # Utilidades
├── pages/          # Páginas de la app
└── test/           # Tests
```
